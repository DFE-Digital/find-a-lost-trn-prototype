## isArray

Checks if a value is classified as an [`Array`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array) object.

Input

```njk
{{ ["england", "scotland", "wales"] | isArray }}
{{ "great britain" | isArray }}
```

Output

```html
true
false
```
