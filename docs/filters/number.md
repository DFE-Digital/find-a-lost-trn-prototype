## isNumber

Checks if a value is classified as a [`Number`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Number) primitive or object.

Input

```njk
{% raw %}{{ 1801 | isNumber }}
{{ "1801" | isNumber }}{% endraw %}
```

Output

```html
true
false
```
